# Base React Project #

A React project served by Webpack and tested with Jest

## Stack ##
Webpack
Babel

Jest
Enzyme
Sinon
Faker
Rosie

React
React Router
PropTypes
QueryString

Stylus
ClassNames
BemClassNames


## Installing ##

```
yarn
```

## Running ##

```
yarn start
```

Using Webpack Dashboard
```
yarn start:dashboard
```

## Testing ##

```
yarn test
```

Watching
```
yarn test:watch
```

Coverage
```
yarn test:coverage
```

Open Coverage Report (osx only)
```
yarn test:coverage:open
```

## Lint ##
```
yarn lint
```
